# Example replicates

This folder contains a small curated subset of `data/generated/` so that
reviewers can execute the analysis scripts without first downloading the
full Figshare archive.

- For every (experiment, model, prompt) combination the **first three**
  replicate files are kept.
- For perception images, up to 12 images per sub-category are kept.
- For the SynAlign pipelines, the first three replicates plus all small
  (< 50 KB) CSV/JSON summaries are kept.

The full dataset (~460 MB) is archived at
<https://doi.org/10.6084/m9.figshare.28910084>.
